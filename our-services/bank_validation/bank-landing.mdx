## Overview
Bank Validation consists of four API endpoints which you can call depending on your requirements:

- **Interactive Validate (v2):** validates the bank account and sort code for any UK bank account.
- **Batch Validate (v1):** allows for bulk validation of UK bank account and sort code details.
- **International Interactive Validate (v1):** validates the bank account and sort code for international bank accounts (excluding the US).
- **Interactive RetrieveBySortcode (v1):** validates using only the sort code for any UK bank account.
